# Calculadora de Irrigação Inteligente

Site simples desenvolvido com HTML, CSS e JavaScript puro.

## Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo
Ajudar no cálculo do consumo de água usado na irrigação de uma plantação, incentivando o uso consciente dos recursos naturais.

## Funcionalidades
- Calcula consumo diário de água.
- Calcula consumo total por período.
- Mostra economia possível com redução de 20%.
- Exibe uma mensagem de orientação sustentável.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Como usar
Abra o arquivo `index.html` no navegador.
